# 🌍 CarbonSense Social Media Features

## Overview
CarbonSense has been transformed into a comprehensive social media platform for climate action, where users can share achievements, connect with like-minded individuals, and stay motivated through community engagement.

## 🚀 Key Social Features

### 1. **Climate Feed** 📱
- **Real-time Social Stream**: Users can share climate achievements, tips, and milestones
- **Achievement Sharing**: Automatic posts when users complete challenges or reach carbon-saving milestones
- **Rich Content**: Support for text, images, achievement badges, and carbon impact displays
- **Interactive Elements**: Like, comment, share, and "inspire" buttons for community engagement
- **Location & Tags**: Users can tag their location and use hashtags for discovery

### 2. **Friends & Network** 👥
- **Follow System**: Users can follow other climate warriors and see their activities
- **Friend Suggestions**: AI-powered recommendations based on similar interests and locations
- **Community Highlights**: Showcase top performers and climate champions
- **User Profiles**: Display carbon savings, level, follower/following counts

### 3. **Social Challenges** 🎯
- **Community Competitions**: Group challenges like "Car-Free Week" with leaderboards
- **Team Challenges**: Corporate or neighborhood-based team competitions
- **Real-time Progress**: Live updates on challenge participation and completion rates
- **Social Motivation**: See friends' progress and encourage each other

### 4. **Notifications & Engagement** 🔔
- **Real-time Notifications**: Likes, comments, follows, and challenge updates
- **Achievement Alerts**: Community celebrates when users reach milestones
- **Friend Activity**: Stay updated on your network's climate actions
- **Smart Filtering**: Prioritize notifications from close connections

### 5. **Communities & Groups** 🏘️
- **Interest-based Communities**: 
  - 🚲 Bike Commuters (2,340 members)
  - 🌱 Urban Gardeners (1,890 members)
  - ⚡ Solar Enthusiasts (956 members)
- **Local Communities**: Connect with neighbors for local climate initiatives
- **Corporate Groups**: Company-wide sustainability challenges and leaderboards

## 🔧 Technical Implementation

### Mobile App (React Native)
```tsx
// Key Social Screens
- SocialFeedScreen.tsx: Main social media feed
- FriendsScreen.tsx: Network management and discovery
- NotificationsScreen.tsx: Social activity notifications
- ChallengesScreen.tsx: Enhanced with social features
```

### Backend API (FastAPI)
```python
# Social Media Endpoints
/api/v1/social/feed - Get personalized social feed
/api/v1/social/posts - Create and manage posts
/api/v1/social/users/{id}/follow - Follow/unfollow users
/api/v1/social/notifications - Real-time notifications
/api/v1/social/trending - Trending topics and hashtags
```

### Web Dashboard (Next.js)
```tsx
// Social Components

```

## 🛠️ Getting Started

### For Developers
```bash
# Clone the repository
git clone https://github.com/oommensy/CarbonSense.git

# Install mobile dependencies
cd mobile && npm install

# Install web dependencies  
cd web && npm install

# Start development servers
npm run dev
```

1. **Join CarbonSense**: Download the app or visit web platform
2. **Complete Your Profile**: Add demographics for personalized recommendations

5. **Join Challenges**: Participate in community challenges for motivation
